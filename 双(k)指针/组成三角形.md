```

    how many  <j, k> such that j < k, and array[k]-array[j] < array[i]，这是一个two difference问题
```


```java
class Solution {
    public int triangleNumber(int[] nums) {
        if (nums == null || nums.length == 0) return 0;
        
        Arrays.sort(nums);
        int count = 0;
        
        for (int k = 2; k <nums.length; k++) {
            int i = 0;
            int j = k - 1;
            
            while (i < j) {
                int sum = nums[i] + nums[j];
                if (sum <= nums[k]) {
                    i++;
                } else {
                    count += (j - i);
                    j--;
                }
            }
        }
        return count;
    }
}
```
