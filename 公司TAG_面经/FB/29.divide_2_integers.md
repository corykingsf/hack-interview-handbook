- 不能用乘除，只能用加减，倍增的思想  
- brute force:每次从被除树中减掉除数，直到减到剩下的数<被除数
  - 优化：二进制的思想(a.k.a.快速幂的思想,倍增的想法)
- 倍增法乘以2在bit里对应的是左移操作

```java
class Solution {
    public int divide(int dividend, int divisor) {
        if (dividend == Integer.MIN_VALUE && divisor == -1) {
			return Integer.MAX_VALUE;
		}
        int sign = (dividend < 0) ^ (divisor < 0) ? -1 : 1;  //处理负数的情况
        long d1 = Long.valueOf(dividend);
        long d2 = Long.valueOf(divisor);
        d1 = d1 > 0 ? d1 :-d1;
        d2 = d2 > 0 ? d2 : -d2;
 
        if (d1 <d2) {
            return 0;
        }

        long ceiling = getCeiling(d1, d2);
        long result = 0;

        while (d1 >= d2) {
            if (ceiling * d2 <= d1) {
                result |= ceiling;
                d1 -= ceiling *  d2;
            }
            ceiling >>= 1;
        }
        return (int)result * sign;
    }
    private long getCeiling(long dividend, long divisor) { //位运算，还可以用倍增法
        long ceiling = 1;
        while (ceiling * divisor <= dividend) {
            ceiling <<= 1;

        }
        return ceiling >> 1;
    }
}
```
