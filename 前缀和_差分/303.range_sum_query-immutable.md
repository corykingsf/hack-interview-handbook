303 [区域和检索-数组不可变](https://leetcode.com/problems/range-sum-query-immutable/)
简单前缀和, 注意数组下标

```java
class NumArray {
    int[] nums;
    int[] s;
    
    public NumArray(int[] nums) {
        this.nums = nums;  
        int len = nums.length;
        s = new int[len + 1];
        
        for (int i = 1; i <= len; i++) {
            s[i] = s[i - 1] + nums[i - 1];
        }
    }
    
    public int sumRange(int left, int right) {
        return s[right +1] - s[left];
    }
}
```
