### 连接问题和路径问题

* 如果我们只想知道a， b两点的连接状态，我们通过求解路径问题来知道连接状态其实消耗了额外的性能


并查集支持的操作：

union(p,q)

isConnected(p,q)  --> find(p) == find(q)  //find返回的是元素的group的ID


并查集可以有不同的底层实现


并查集接口:


```java
public interface UF {
    int getSize();
    boolean isConnected(int p, int q);
    void unionElements(int p, int q);
}
```


quick find下的find() 时间是O(1), union的时间是O(n)
