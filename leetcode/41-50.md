50 快速幂
-

- solution 1: O(n)的时间通过循环将n个x乘起来

```java
class Solution {
    public double myPow(double x, int n) {  
        double res = 1.0;
        boolean isMinus = n < 0;
        long b = Math.abs((long)n);
        
        while (b > 0) {
            if ((b & 1) == 1) res *= x;  // 从低位到高位逐位判断是不是1，如果是1，就把x的幂乘上去
            x *= x;   //边算边预处理x的幂次
            b >>= 1;   // 把最低位踢出去
        }
        
        if (isMinus) res = 1 / res;
        return res;
    }
}


```
