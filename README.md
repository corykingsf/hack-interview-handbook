# hack-interview-handbook

### 数据驱动算法
先学线性数据结构,然后学习其上各种灵活的算法
了解树，图等结果，然后学习递归、搜索等算法
形成集合、维度、状态空间等概念，然后学习动态规划等高级算法

抛开数据的支持，算法是假大空
脑子里有了数据结构的具体形状，算法将会更加形象，容易理解
